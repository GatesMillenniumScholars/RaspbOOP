Detailed Description

Server class used to receive Commands.

The Server class uses UDP Sockets to exchange information as encoded and decoded by the Command class and the Raspboop Transmission Protocol.

The Server class also provides automatic discovery functionality through a Multicast sockets. Invoke EnableAutodiscovery() to start Multicasting.
