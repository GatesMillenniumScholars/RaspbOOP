Detailed Description

Passive infrared sensor for motion detection.

The HCSR501 is a passive infrared sensor used for motion detection applications. There are 5 main pins that you will need to be familiar with:

  • 5V input
  • Signal
  • Gnd
  • retriggering pin
  • non-retriggering pin

Helpful information

There are two modes to the sensor: retriggering, and non-retriggering. In retriggering mode, signal outputs high as long as motion is detected. In non-retriggering mode, signal will output as HIGH only every second or so that motion is detected.
