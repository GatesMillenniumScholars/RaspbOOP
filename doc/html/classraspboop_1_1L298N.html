Detailed Description

L298N Motor controller.

This class is meant to support the L298N motor controller board. In most cases, the Enable A, B, and 5V pins are connected by default through jumpers on most model #236 board.

Helpful information

  • This board is able to control two motors at the same time.
  • Do not input a voltage into the 5V terminal
    • If the voltage in the 12V terminal is greater than 12V, then you can use the 5V terminal as a power source.

void raspboop::L298N::UseSoftPWM ( )

Sets the input pins to software PWM mode.

Uses wiringPi's software PWM library to set the pins provided in the Create() factory method to a software PWM mode

Note: Software PWM operates on seperate threads, and each thread consumes about 0.5% CPU, according to the wiringPi reference.
